# ✈️ U.S. Airline Performance & Delay Analysis

## 📌 Project Overview

Developed an end-to-end Aviation Analytics solution using **SQL Server** and **Power BI** to analyze **5.8M+ U.S. domestic flight records**, identify delay and cancellation drivers, benchmark airline and airport performance, and provide actionable business recommendations.

---

## 🎯 Business Objective

- Analyze flight delays and cancellations.
- Benchmark airline operational performance.
- Identify airport bottlenecks.
- Evaluate route reliability.
- Analyze temporal trends affecting flight operations.

---

## 🛠️ Tech Stack

- SQL Server
- Power BI
- Excel

---

## 📊 Key Metrics

- **5.8M+ Flights Analyzed**
- **628 Airports**
- **14 Airlines**
- **8,609 Routes**
- **82.14% OTP Rate**
- **90K+ Cancelled Flights**
- **1.54% Cancellation Rate**

## 🔍 Key Findings

- Late Aircraft Delays contributed **416K+ delay hours**, making them the largest delay driver.
- Airline-related factors accounted for **54% of cancellations**.
- Hawaiian Airlines achieved the highest OTP rate (**89.47%**).
- Several regional airports recorded cancellation rates above **10%**.
- Delays increased significantly during evening operating hours.

---

## 💡 Recommendations

- Improve aircraft turnaround efficiency.
- Optimize scheduling for high-risk routes.
- Focus operational improvements on high-delay airports.
- Use historical trends for predictive capacity planning.
